# Coding Eval Baselines

This directory is reserved for raw OmniFusion coding-eval run outputs and
baseline comparisons. No provider benchmark result has been run or claimed yet.

Required provenance for a real baseline:

- OmniFusion commit SHA.
- Aider version and exact driver config.
- Task suite file checksum.
- Provider/model identifiers.
- Raw pass/fail, wall time, and cost output.
- Confidence interval and cost-normalized summary for full-suite runs.
